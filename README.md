# Rest'O Mermoz 

Application e-commerce de livraison de nourriture fullstack.

## Technologies
- Frontend : React, Context API, React Router
- Backend : Node.js, Express, MongoDB
- Paiement : Stripe
- Auth : JWT

## Lancer le projet

### Backend
cd Backend
npm install
npm run server

### Frontend
cd Frontend
npm install
npm run dev

### Admin
cd Admin
npm install
npm run dev
